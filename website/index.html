Summary: Lattitude vs. X

This project aimed to analyze how weather changes as you get closer to the equator. We first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities to accomplish this analysis.After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis and explanations and descriptions of any trends and correlations witnessed.
